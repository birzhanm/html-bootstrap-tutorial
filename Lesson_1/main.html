Question: What is HTML?

Formally, HTML is a Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.
Informally, HTML is an enriched text file which can be displayed on browser.

Question 2: How to write a simple HTML file?

Remember we have said that HTML files are just text files, so it means that they can be written on Notepad. The only thing to keep in mind is to add an extension '.html'. However, most of IDEs, say Atom, provide many helpful tools in writing HTML documents such as tags autocompletion, highlighting and so on. So one might find it easier to write HTML documents using an IDE.
